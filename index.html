Início Jogar

🧩 Sobre o Jogo

Este jogo foi desenvolvido como uma ferramenta assistiva para pessoas no espectro autista, estimulando memória visual, concentração e cognição de forma lúdica. As cores são suaves, os sons são tranquilos e a interface foi pensada para conforto e bem-estar.

🎮 Jogar Agora

❓ Como Jogar

  1. ➡️ Clique em "Jogar Agora" ou no topo da página.
  2. 🃏 Escolha duas cartas para virar.
  3. 🔍 Se forem iguais, permanecem abertas.
  4. 🔄 Se forem diferentes, voltam para a posição inicial.
  5. 🎯 O objetivo é encontrar todos os pares.

Dica: Jogue no seu ritmo, sem pressão de tempo.
